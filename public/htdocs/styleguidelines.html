HTML Components and Styles

This page aims to provide detail on the standard HTML components and layouts used to build pages for the childsifoundation.org sites

It is meant to act as a shared and ongoing document to ease the creation of pages

1. Grid system

This is a simple grid.
There are five columns
Each is 175px wide.
You wrap each .col in a .row
This will eventually support responsive versions.
You can also span columns by using .w2 .w3 .w4 .25 applied to .col divs
The cols should always add up to five though. But they don't have to.
Each .row has a 15px bottom margin, which you can remove using .unspaced
