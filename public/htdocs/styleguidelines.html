1. Grid system

This is a simple grid.
There are five columns
Each is 175px wide.
You wrap each .col in a .row
This will eventually support responsive versions.
You can also span columns by using .w2 .w3 .w4 applied to .col divs
The cols should always add up to five though. But they don't have to.
Each .row has a 15px bottom margin, which you can remove using .unspaced
